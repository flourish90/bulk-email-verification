Bulk-Verify Email Adresses using reacher.email

Instructions

  1. Click "Select CSV-File" and choose the CSV-file from your system. The file must contain only Email adresses, one adress per line.
  2. Sample file can be found here.
  3. Click "Verify Emails"
  4. Emails from the file will be listest below "Results"
  5. After a few seconds, verification results will be shown next to each adress.
  6. Note that for privacy reasons, the csv-file will not be uploaded. It will stay safely on your system at all times.
